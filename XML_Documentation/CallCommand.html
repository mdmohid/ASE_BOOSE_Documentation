Class CallCommand
Represents a command that invokes a user-defined method in the BOOSE application.
Syntax
C#
    internal class CallCommand : Command
Remarks
The CallCommand is responsible for:
  • Validating method existence
  • Passing arguments to the method
  • Managing method scope
  • Capturing and returning the method's result

Method Execute
Executes the method call.
Syntax
C#
    public override void Execute()
Exceptions
Exception Condition
Exception Thrown when the argument count does not match the method definition.
Remarks
This method performs the following steps:
  1. Checks whether the method exists
  2. Validates argument count
  3. Creates a new execution scope
  4. Declares parameters and executes the method body
  5. Retrieves and stores the return value
