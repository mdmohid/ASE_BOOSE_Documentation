Class AssignmentCommand
Represents a command that assigns a computed value to an existing variable.
Syntax
C#
    internal class AssignmentCommand : ICommand
Remarks
This command evaluates an expression based on the variable's existing type (integer, real, or boolean) and updates the variable value in the current scope.
