Class ExpressionEvaluator
Provides static helper methods for evaluating expressions used in the BOOSE command language.
Syntax
C#
    public static class ExpressionEvaluator
Remarks
This evaluator supports:
  • Integer arithmetic expressions
  • Real (double) arithmetic expressions
  • Boolean expressions including logical operators

Method EvaluateInt
Evaluates an expression and returns its integer value.
Syntax
C#
    public static int EvaluateInt(string expression)
Parameters
expression
The expression to evaluate.
Return Value
The evaluated integer result.
Remarks
Internally evaluates the expression as a real number and casts the result to an integer.

Method EvaluateReal
Evaluates an arithmetic expression and returns its real value.
Syntax
C#
    public static double EvaluateReal(string expression)
Parameters
expression
The arithmetic expression to evaluate.
Return Value
The evaluated double result.
Exceptions
Exception Condition
Exception Thrown when the expression is empty, invalid, or contains unsupported operators.
Remarks
Supported operators are: +, -, *, and /. Variables referenced in the expression are resolved using AppSingleton.

Method ResolveValue
Resolves a token into a numeric value.
Syntax
C#
    private static double ResolveValue(string token)
Parameters
token
The token to resolve.
Return Value
The numeric value of the token.
Exceptions
Exception Condition
Exception Thrown if the token is empty or does not represent a numeric literal or variable.
Remarks
The token may be a numeric literal or the name of a variable stored in the current application scope.

Method EvaluateBoolean
Evaluates a boolean expression.
Syntax
C#
    public static bool EvaluateBoolean(string expression)
Parameters
expression
The boolean expression to evaluate.
Return Value
The evaluated boolean result.
Exceptions
Exception Condition
Exception Thrown when the expression cannot be evaluated as a boolean.
Remarks
Supported boolean features:
  • Boolean literals: true, false
  • Logical AND: &&
  • Logical OR: ||
  • Logical NOT: !
  • Boolean variables
