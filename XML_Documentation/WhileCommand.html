Class WhileCommand
Command that represents a while loop.
Syntax
C#
    internal class WhileCommand : ICommand
Remarks
The while loop repeatedly executes the commands in bodyCommands as long as the conditionExpr evaluates to true. The condition supports simple integer comparisons, e.g., "x < 10". A new scope is pushed for each iteration to isolate loop variables.
