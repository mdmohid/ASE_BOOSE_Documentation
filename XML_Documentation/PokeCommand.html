Class PokeCommand
Command to write a value into an array at a specified index.
Syntax
C#
    internal class PokeCommand : ICommand
Remarks
The PokeCommand evaluates an index expression and a value expression, then assigns the evaluated value to the specified position in the array. The array must already exist and be of type Double[].
