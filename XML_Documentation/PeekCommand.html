Class PeekCommand
Command to read a value from an array and assign it to a variable.
Syntax
C#
    internal class PeekCommand : ICommand
Remarks
The PeekCommand retrieves a value from a specified array index and stores the retrieved value into a target variable. The index may be provided as a literal, variable, or expression evaluated at runtime.
