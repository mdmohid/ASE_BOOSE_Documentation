Class DeclareVariableCommand
Command responsible for declaring variables in the application.
Syntax
C#
    internal class DeclareVariableCommand : ICommand
Remarks
Supports declaration of the following types:
  • int
  • real
  • boolean
  • array
Variables may be declared with or without an initial value.

Exception Condition
Exception Thrown when the declaration format is invalid.
Remarks

Array declarations support the following formats:

  • array name size
  • array int name size
  • array real name size

Scalar variables may optionally include an initialization expression.
