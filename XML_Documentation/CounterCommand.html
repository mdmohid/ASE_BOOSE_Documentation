Class CounterCommand
CounterCommand increments an integer variable by 1 each time it is executed. Used for testing control-flow commands like IfCommand, WhileCommand, and ForCommand. This version automatically declares the variable if it does not exist in the current scope, making unit tests safer when running multiple tests sequentially or together.

C#
