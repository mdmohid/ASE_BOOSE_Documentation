Class MethodCommand
Represents a method declaration command.
Syntax
C#
    internal class MethodCommand : Command
Remarks
This command registers a user-defined method in the application without executing its body. Method execution is handled separately by the CallCommand.

Method Execute
Executes the method declaration.
Syntax
C#
    public override void Execute()
Remarks
The method is registered in the global method table. If a method with the same name already exists, it is overwritten. The method body is not executed during this step.
